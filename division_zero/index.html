I mean yeah, I guess 0 divided by something makes sense, because you just have nothing to begin with. That rule stays established. But I really wanted to dive into the other side of it.

Multiplication by 0 and Multiplicative Inverses

In rudimentary algebra, we can define a division problem as x=a/bx = a/bx=a/b (e.g., 3=6/23 = 6/23=6/2). Now, in our case, if we take an arbitrary real number (let's say 6) we can set it up as x=6/0x = 6/0x=6/0. Based on the rules of algebra, this technically should also be able to be written as:

0⋅x=60 \cdot x = 60⋅x=6

Is there any value of xxx that satisfies this? Not that I know of!

A multiplicative inverse of a number is one that you can multiply by the original number to get 1. For example, 2's multiplicative inverse is 1/21/21/2, 4 is 1/41/41/4, and so on. 0… doesn't really have one.

Defining the Limit (or lack of one)

Say we have the function f(x)=1/xf(x) = 1/xf(x)=1/x, and we want to find the limit as xxx approaches 0. Usually, we do this by approaching the desired value of xxx from both sides individually to see if they are equal (i.e., if and only if the left-hand and right-hand limits are consistent).

If we apply this here, as we approach 0 from the left, we approach negative infinity (−∞-\infty−∞); when we approach 0 from the right, we approach positive infinity (+∞+\infty+∞). Because the two sides don't meet, the limit does not exist.
